## Code Detection Toggle

The main toggle lets you easily turn on and off the code detection feature in its entireity.

<Steps>
  <Step title="Access & Use Dashboard" icon="fingerprint" iconType="solid" stepNumber={1} titleSize="h3">
    Follow the steps in [Accessing the Dashboard](/dashboard/access) to login to the dashboard, then visit [Using the Dashboard](/dashboard/use) to select a server and start configuring.
  </Step>
  <Step title="Locate & Enable / Disable" icon="binoculars" iconType="solid" stepNumber={2} titleSize="h3">
    On the server page, find the 'Code Detection Config' box. It should be one above the bottom, regardless of mobile or PC. Find the 'Code Detection' box. Click the enable or disable button as needed. The current setup will be auto selected on page load.
  </Step>
</Steps>

## Detection Result Limit

The detection result limit controls how many codes to show fetched info for before the rest are truncated. The default is 3, but it can be set to whatever you want.

<Steps>
  <Step title="Access & Use Dashboard" icon="fingerprint" iconType="solid" stepNumber={1} titleSize="h3">
    Follow the steps in [Accessing the Dashboard](/dashboard/access) to login to the dashboard, then visit [Using the Dashboard](/dashboard/use) to select a server and start configuring.
  </Step>
  <Step title="Locate & Enable / Disable" icon="binoculars" iconType="solid" stepNumber={2} titleSize="h3">
    On the server page, find the 'Code Detection Config' box. It should be one above the bottom, regardless of mobile or PC. Find the 'Detection Result Limit' box. Click the enable or disable button as needed. The current setup will be auto selected on page load.
  </Step>
</Steps>
