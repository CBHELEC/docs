To find out what skullboard is, go to [Skullboard](/features/skullboard).

<Steps>
  <Step title="Access & Use Dashboard" icon="fingerprint" iconType="solid" stepNumber={1} titleSize="h2">
    Follow the steps in [Accessing the Dashboard](/dashboard/access) to login to the dashboard, then visit [Using the Dashboard](/dashboard/use) to select a server and start configuring.
  </Step>
  <Step title="Locate Skullboard Settings" icon="binoculars" iconType="solid" stepNumber={2} titleSize="h2">
    On the server page, find the skullboard configuration box. It should be the very first one shown, regardless of mobile or PC. Find the 'Split Overlength Messages' box. The current settings will be auto selected on page load.
  </Step>
  <Step title="Configure Skullboard" icon="wrench" iconType="solid" stepNumber={3} titleSize="h2">
    There is only one option to configure, in the form of a dropdown. This shows all of the channels in your server, in order, with any uncategorised at the top.
