## Log / Queue Channel

<Steps>
  <Step title="Access & Use Dashboard" icon="fingerprint" iconType="solid" stepNumber={1} titleSize="h3">
    Follow the steps in [Accessing the Dashboard](/dashboard/access) to login to the dashboard, then visit [Using the Dashboard](/dashboard/use) to select a server and start configuring.
  </Step>
  <Step title="Locate & Toggle" icon="sliders" iconType="regular" stepNumber={2} titleSize="h3">
    On the server page, find the Verification area. It should be the last section, regardless of on PC or mobile. The current settings will be auto selected on page load. Within this, find the 'Log/Queue Channel' dropdown box, showing all of the channels in your server, in order, with any uncategorised at the top. To <u>disable</u> this setting, open the dropdown and select the 'ðŸš« None' option at the top. To set a channel, open the dropdown, and pick any channel shown, and press the 'Set Channel' button.

    <Warning>
      When this function is set to 'ðŸš« None', verifications will not be sent, and you won't be able to see / approve / deny them.
    </Warning>
  </Step>
</Steps>

## Fasttrack Status

<Steps>
  <Step title="Access & Use Dashboard" icon="fingerprint" iconType="solid" stepNumber={1} titleSize="h3">
    Follow the steps in [Accessing the Dashboard](/dashboard/access) to login to the dashboard, then visit [Using the Dashboard](/dashboard/use) to select a server and start configuring.
  </Step>
  <Step title="Locate & Toggle" icon="sliders" iconType="regular" stepNumber={2} titleSize="h3">
    On the server page, find the Verification area. It should be the last section, regardless of on PC or mobile. The current settings will be auto selected on page load. Within this, find the 'Fasttrack Status' toggles. To <u>disable</u> this setting, click the 'Disable' button. To <u>enable</u> this setting, click the 'Enable' button. The new choice will be auto-saved and highlighted accordingly.
  </Step>
</Steps>

## Verified Role

<Steps>
  <Step title="Access & Use Dashboard" icon="fingerprint" iconType="solid" stepNumber={1} titleSize="h3">
    Follow the steps in [Accessing the Dashboard](/dashboard/access) to login to the dashboard, then visit [Using the Dashboard](/dashboard/use) to select a server and start configuring.
  </Step>
  <Step title="Locate & Toggle" icon="sliders" iconType="regular" stepNumber={2} titleSize="h3">
    <Info>
      For this to work, the role needs to exist inside the Discord server.
    </Info>
    On the server page, find the Verification area. It should be the last section, regardless of on PC or mobile. The current settings will be auto selected on page load. Within this, find the 'Verified Role' dropdown box, showing all of the roles in your server, in order. To <u>disable</u> this setting, open the dropdown and select the 'ðŸš« None' option at the top. To set a channel, open the dropdown, and pick any channel shown, and press the 'Set Channel' button.

    To remove a role, click the <Icon icon="circle-xmark" iconType="solid" size={20} /> next to the role name.
  </Step>
</Steps>
