# Command Reference

This is the practical command map for players.

![Placeholder - Command Categories Graphic](./images/placeholder-command-categories.png)

## General

- `/discaching help` - Open help menu.
- `/discaching profile` - View your stats and progress.
- `/discaching inventory` - Browse inventory categories.
- `/discaching balance` - View current G$.
- `/discaching daily` - Claim daily G$ reward (G$25).
- `/discaching quests` - View daily quests and claim quest crates.
- `/discaching shop` - Open shop and buy/sell items.
- `/discaching leaderboard` - View top players by category.
- `/discaching gift [user] [amount]` - Send G$ to another player.
- `/discaching open [Vote Crate|Quest Crate] [amount]` - Open crates.
- `/discaching credits` - Project credits.

## Caching Gameplay

- `/discaching hide` - Start or continue hide setup.
- `/discaching find` - Log found/DNF/note on a cache.
- `/discaching nearby` - Show nearest caches.
- `/discaching cache_info [cache_id]` - Detailed cache info.
- `/discaching cache_finds [cache_id]` - View cache logs.
- `/discaching locations [Rural|City]` - Region list and flavor locations.

## Travel and Position

- `/discaching travel [cache_id]` - Travel directly, or open map if omitted.
- `/discaching travel cancel_current:true` - Cancel active travel.
- `/discaching location` - Current coordinates and travel status.
- `/discaching map` - Map with your location and published caches.
- `/discaching distance_to_cache [cache_id]` - Distance and estimate.

## Trackables

- `/discaching trackable_activate [private_code]`
- `/discaching trackable_view [public_code]`
- `/discaching trackable_grab [public_code]`
- `/discaching trackable_discover [private_code] [log]`

## Notes

- Most game commands are ephemeral responses.
- Cache IDs support autocomplete in many commands.
- Admin-only commands are in `/discaching_admin`.
