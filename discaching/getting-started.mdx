Discaching is a geocaching-style game inside Discord. You earn money, travel, hide caches, find caches, and build your profile.

## First Time Setup

When you run your first `/discaching` command, you automatically get some stuff.

- Starting balance: **G\$100**
- Starting favorite points owned: **10**
- Starting coordinates: **27.07, -75.91**

## Your First 10 Minutes

1. Run `/discaching help` or visit [here](/discaching/commands) to see commands.
2. Run `/discaching daily` for your daily G\$25.
3. Run `/discaching find` and find GX1.
4. Run `/discaching map` to open the map and see the caches.
5. Run `/discaching cache_info [GX_]` to see info about a cache,
6. Run `/discaching travel [GX_]` to go to the cache.
6. Run `/discaching find` once you have travelled, to find it.

## Core Rules

- You cannot find your own cache.
- You cannot log a cache as found without a usable writing instrument, unless the cache has one inside it.
- If you are currently traveling, you must wait for arrival (or cancel travel) before starting another trip.
- If your account is suspended, game commands are blocked.
- All terms of service apply. We recommend reading them before playing. If you don't, you are still liable for consequences.

## Daily Reset Behavior

- `/discaching daily` can be claimed once per day.
- Daily quests reset at UTC day rollover.
