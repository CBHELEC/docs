# Getting Started

Discaching is a geocaching-style game inside Discord. You earn money, travel, hide caches, find caches, and build your profile.

![Placeholder - New Player Overview](./images/placeholder-new-player-overview.png)

## First Time Setup

When you run your first `/discaching` command, the game creates your profile automatically.

Default values:
- Starting balance: **G$100**
- Starting favorite points owned: **10**
- Starting coordinates: **27.07, -75.91**

## Your First 10 Minutes

1. Run `/discaching help`.
2. Run `/discaching daily` for your daily G$25.
3. Open `/discaching shop` and buy at least:
   - 1 writing instrument (pen or pencil)
   - 1 container
4. Run `/discaching travel` to open the map and move near a cache area.
5. Run `/discaching nearby` and then `/discaching cache_info [cache_id]`.
6. Run `/discaching find` once you are close enough.

![Placeholder - First Session Checklist](./images/placeholder-first-session-checklist.png)

## Core Rules You Should Know Early

- You cannot find your own cache.
- You cannot log a found cache without a usable writing instrument unless the cache already has one.
- If you are currently traveling, you must wait for arrival (or cancel travel) before starting another trip.
- If your account is suspended, game commands are blocked.

## Daily Reset Behavior

- `/discaching daily` can be claimed once per date.
- Daily quests reset at UTC day rollover.
