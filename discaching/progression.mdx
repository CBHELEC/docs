## Daily Quests

Use `/discaching quests`.

How it works:

- You get 4 quests per day.
- Typical mix: 2 easy, 1 medium, 1 hard.
- Completing quests gives claimable **Quest Crates**.
- Claiming is manual via quest UI button.

Quest types include:

- Finds
- Travel distance
- Spending
- Daily claim
- Favorite point giving
- Gifting
- Trackable actions
- Shop purchases
- Hides
- Logs created
- Writing instrument usage
- Vehicle usage
- Location visits
- Inventory count
- FTF
- Sticker placing/collecting

## Collectible Stickers

- Buy from shop.
- Place into caches.
- Collect from caches.
- Completing full sticker collection grants one-time bonus: **G\$1,750**.

## Achievement Stickers

Awarded on milestones for categories like:

- Finds
- Hides
- FTF
- Total spending
- Balance
- Favorite points received
- Trackables owned/activated/discovered
- Pencil breaks
