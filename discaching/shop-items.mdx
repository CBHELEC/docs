# Shop and Items

Use `/discaching shop` to buy and sell game items.

![Placeholder - Shop Main Screen](./images/placeholder-shop-main.png)

## Currency

- Main currency: **G$**
- Starting balance: **G$100**
- Daily gain: **G$25** via `/discaching daily`

## Shop Categories

- Writing Instruments
- Container Accessories
- Containers
- Transport
- Trackables
- Stickers
- Swag

## Why item choices matter

### Writing Instruments

Used for Found It logs unless cache already has one.
Each instrument has use count.

### Containers

Used to create hides.
Have supported sizes and optional modifiers (size/color/log).

### Transport

Used for travel speed and range.
Some items have persistent usage (vehicles), others are consumed (tickets).

### Trackables

Purchased as item type that generates a public/private code pair.

### Stickers and Swag

Used for collecting, placing, and cache exchange interactions.

## Selling Items

- Sell flow is available from the shop UI.
- Sale value is **50%** of item price (integer floor).
- You can sell inventory items and collectible stickers.

## Crates

Use `/discaching open` with:
- Vote Crate
- Quest Crate

Crates give:
- Random money
- Random items
- Sometimes trackables (with generated codes)
